Git GitHub >>>>> Todos os arquivos listados no arquivo .gitignore não serão monitorados pelo GitHub * Comandos / git init = Inicializa o repositorio na pasta selecionada / git status = checa o estado dos arquivos na pasta inicializada / git add # = adiciona o arquivo especificado para ser comitado / git add . = adiciona todos os aquivos da pasta para serem comitados / git add remote $ @ = cria um repositorio remoto / git commit -m "$" = salva as modificações feitas nos arquivos (Recomendado uso de descrição da alteração feita) / git log = lista todas as alterações feitas / git log --online = lista todas as alterações feitas mostrando em uma linha / git log -p = lista todas as alterações feitas em mais detalhes / git init --bare = cria um repositorio puro, que conterá somente as alterações feitas, não os arquivos / git remote = lista todos os repositorios remotos reconhecidos pelo PC / git remote add & @ = adiciona um servidor remoto / git clone @ $ = clona um repositorio remoto com um novo nome / git rename & & = renomeia o servidor localmente / git push & % = envia o conteudo do repositorio para o servidor remoto / git push -u & % = envia o conteudo do repositorio para o servidor remoto especificando para qual servidor e brench / git pull & % = trás o conteudo do repositorio remoto para a pasta / git branch $ = cria um novo branch / git checkout % = muda o branch em que se está / git checkout -b $ = cria um novo branch e muda o branch em que se está / git merge % = une o branch atual com o branch especificado (use o codigo [:X] para finalizar) / git rebase % = atualiza o branch atual com todas as alterações do branch especificado * Configuração / git config --local = modificações serão feitas no projeto / git config --global = modificações serão feitas no computador / git config --local user.name "$" = modifica o nome do usuario * Dicionario / cd = Navegação entre pastas / cd .. = Subur uma pasta / ls = lista todos os arquivos da pasta / mkdir $ = cria uma nova pasta com o nome selecionado * Lista de siglas do texto / # = Nome do arquivo / $ = Texto / @ = Caminho do arquivo/servidor / & = Nome do servidor / % = Nome do branch
