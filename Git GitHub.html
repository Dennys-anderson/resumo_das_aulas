Git GitHub >>>>> Todos os arquivos listados no arquivo .gitignore não serão monitorados pelo GitHub * Comandos / git init = Inicializa o repositorio na pasta selecionada / git status = checa o estado dos arquivos na pasta inicializada / git add # = adiciona o arquivo especificado para ser comitado / git add . = adiciona todos os aquivos da pasta para serem comitados / git commit -m "$" = salva as modificações feitas nos arquivos (Recomendado uso de descrição da alteração feita) / git log = lista todas as alterações feitas / git log --online = lista todas as alterações feitas mostrando em uma linha / git log -p = lista todas as alterações feitas em mais detalhes / git init --bare = cria um repositorio puro, que conterá somente as alterações feitas, não os arquivos * Configuração / git config --local = modificações serão feitas no projeto / git config --global = modificações serão feitas no computador / git config --local user.name "$" = modifica o nome do usuario * Dicionario / cd = Navegação entre pastas / cd .. = Subur uma pasta / mkdir $ = cria uma nova pasta com o nome selecionado * Lista de siglas do texto / # = Nome do arquivo / $ = Texto
